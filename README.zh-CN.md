👉 [预览](https://cnode.funn1est.cn/)

## 技术栈

- React
- Redux
- React Router
- Immutable
- Ant Design
- webpack 4
- Babel
- Jest + Enzyme + Sinon
- ESLint + Prettier + Stylelint
- husky + lint-staged

## 快速开始

```bash
$ git clone
$ cd react-cnode
$ yarn # Install dependencies
$ yarn dev # Development
$ yarn prod # Production
$ yarn test # Test
$ yarn test:u # Update snapshots
```
